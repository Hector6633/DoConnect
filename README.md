🩺 Doconnect – Doctor Appointment Booking System
Doconnect is a web-based doctor appointment booking platform that allows users to easily connect with healthcare professionals, book appointments, and manage their medical schedules efficiently.

🚀 Features
📅 Easy Doctor Appointment Booking
👨‍⚕️ Multiple Healthcare Services (Consultation, Checkups, Tests, etc.)
🔔 Appointment Notifications & Reminders
🧾 Patient Data Management
🖥️ User-Friendly Interface
⚡ Optional Live Reload for faster development
🛠️ Tech Stack
Backend: Python, Django
Frontend: HTML5, CSS3, Bootstrap, JavaScript
Database: MySQL / SQLite
Tools: Git, Virtual Environment
⚙️ Installation & Setup
Follow the steps below to run the project locally:

Step 1: Create Virtual Environment
Step 2: Activate Virtual Environment Windows: your_virtual_environment_name\Scripts\activate

Mac/Linux: source your_virtual_environment_name/bin/activate

Step 3: Install Dependencies pip install -r requirements.txt

Step 4: Run the Development Server python manage.py runserver

Step 5: (Optional) Run Live Reload Server python manage.py livereload

⚠️ Note: Live reload is optional and used only for improving development speed.
